# 키오스크 학습 시뮬레이션 앱

React Native로 개발된 키오스크 사용법을 학습할 수 있는 시뮬레이션 앱입니다.

## 🎯 주요 기능

### 1. 메인 화면
- 앱 시작 시 "키오스크 학습을 시작하세요" 메시지 전달
- 학습 과정 소개 및 특징 안내

### 2. 학습 레벨 선택
- **초급**: 기본 메뉴 선택 및 결제 방법 (5-10분)
- **중급**: 옵션 선택 및 할인 혜택 활용 (10-15분)
- **고급**: 복합 주문 및 문제 해결 방법 (15-20분)

### 3. 메뉴 선택
- 다양한 메뉴 카테고리 (음료, 디저트, 음식)
- 메뉴 정보: 이미지, 이름, 가격, 설명
- 장바구니 기능 및 선택된 메뉴 관리

### 4. 결제 시스템
- 카드 결제 및 현금 결제 지원
- 주문 요약 및 결제 금액 확인
- 결제 과정 안내 및 시뮬레이션

## 🚀 기술 스택

- **React Native**: 크로스 플랫폼 모바일 앱 개발
- **React Navigation**: 화면 간 네비게이션
- **Linear Gradient**: 그라데이션 배경 효과
- **Vector Icons**: 아이콘 및 이모지 사용

## 📱 화면 구성

1. **WelcomeScreen**: 앱 시작 및 환영 메시지
2. **LevelSelectionScreen**: 학습 레벨 선택
3. **BeginnerCourseScreen**: 초급 과정 소개
4. **IntermediateCourseScreen**: 중급 과정 소개
5. **AdvancedCourseScreen**: 고급 과정 소개
6. **MenuSelectionScreen**: 메뉴 선택 및 장바구니
7. **PaymentScreen**: 결제 방법 선택 및 처리

## 🎨 UI/UX 특징

- **직관적인 디자인**: 사용자가 쉽게 이해할 수 있는 인터페이스
- **단계별 안내**: 각 과정별 상세한 설명과 예상 학습 시간 제공
- **시각적 피드백**: 선택 상태, 진행 상황을 명확하게 표시
- **반응형 레이아웃**: 다양한 화면 크기에 대응하는 디자인

## 📋 설치 및 실행

### 1. 의존성 설치
```bash
npm install
```

### 2. iOS 의존성 설치 (iOS 개발 시)
```bash
cd ios && pod install && cd ..
```

### 3. 앱 실행
```bash
# Android
npm run android

# iOS
npm run ios

# Metro 서버 시작
npm start
```
